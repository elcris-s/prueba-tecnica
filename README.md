## Tecnologias utilizadas 🛠️

* C#
* MVC
* ASP.NET Core
* [Bootstrap](https://getbootstrap.com/)

* [DataTables](https://datatables.net/) Para el uso de la tabla que muestra todos los libros y se tenga paginacion, filtros y busqueda
